Rebel Technology is releasing the OWL Modular Eurorack synth in August.  It supports patches for the OWL programmable effects pedal and can be extended with user code.  (See also GitHub for source code.)

And Novation is exploring open firmware with the Launchpad Pro grid controller.

Not sure how many people will make use of these programming interfaces but it’s great to see products being designed with user modification in mind.
