Patterning is an iPad drum machine from Olympia Noise Co that implements a unique sequencing interface organized around concentric circles.  It's sophisticated yet accessible and lets you create dynamic patterns with a surprisingly minimal amount of effort.

In a previous post I wrote about how to configure Patterning to receive clock data from Bitwig Studio, and in the video below, I demonstrate how to use Patterning as a MIDI controller to sequence a drum machine in Bitwig.  Of course, you're not limited to using Patterning for drum sounds but that's what I've done here.

For MIDI I'm using MusicIO, which is a great little app that lets you transfer MIDI and audio data between a Mac/Win computer and iOS device using a USB to lightning cable.  It's the simplest and cheapest way to get an iOS device integrated with a computer for music production.
