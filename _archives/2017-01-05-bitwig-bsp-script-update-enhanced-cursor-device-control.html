I made a few small tweaks to the Bitwig Beatstep Pro script I released a few weeks ago.

  1. The encoders now control parameters and macros for the cursor device of the selected track instead of the primary device.  Since you can navigate the selected track's device chain using steps #3 and #4 in control mode I thought it'd be nice to be able to control macros/params for all the devices in a chain.
  2. Step #9 will toggle the cursor device's enabled state.  This makes it easy to turn devices off and on right from the BSP.

For more info and installation steps, read the original post.
